# Estimated Egress Fee

The Estimated Egress Fee shows the estimated internet egress cost for the discovered data footprint.

**What you will see**

In `egress.html`, this section shows:

- the estimated egress fee
- a breakdown by resource type
- a free-tier state when the footprint is within the provider allowance

If some resources cannot be sized, the estimate is shown as a lower bound.

**Pricing**

The estimate is based on public cloud provider list prices for internet egress and, where applicable, archive retrieval.

The exact prices used for the assessment are listed in the PDF appendix together with the source links.

List prices at the time of the assessment; excludes request/transaction costs and cross-region traffic.
